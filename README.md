# sample_mushroom_app

## キノコの画像を読み込んで種類を判定します

キノコ狩りをする際に食毒の参考情報として活用します。

モデルは訓練済みのresnet101を利用して、キノコの画像で追加訓練したものです。

現状2種類のキノコの判定ですが、mushroom_datasetに訓練画像、mushroom_classesに名称を追加することで判定対象を増やすことができます。
